Документация UI Kit

Страница содержит описание компонентов, примеры использования и БЭМ-классы для подключения в итоговом проекте.

Button

Кнопки имеют варианты primary, secondary и disabled, три размера и состояния hover, active, focus.

<button class="button button--primary button--medium" type="button">Primary</button>

Input, Dropdown, Checkbox и Radio

Поля поддерживают нормальное состояние, focus, error-сообщение и иконку внутри input.

Элементы выбора

Card

Карточка содержит заголовок, изображение и текст. Модификаторы задают наличие изображения и размеры small, medium, large.

Small card

Компактная карточка для коротких списков.

Medium without image

Вариант без изображения подходит для текстовых блоков и уведомлений.

Large card

Широкая карточка для промо-блоков, статей или ключевых возможностей продукта.

Layout

Класс container центрирует контент, stack реализует flex-выравнивание, grid--cards создает адаптивную сетку карточек.

Блок Назначение
.container Центральное выравнивание контента с максимальной шириной.
.stack Flexbox-группа для кнопок и inline-элементов.
.grid.grid--cards CSS Grid для галерей и списков карточек.
